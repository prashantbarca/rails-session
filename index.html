    MVC Example

    Model

  • Maps a table in the database with a ruby object
  • Validates your data before saving them
  • Here you manipulate your data
  • Here you add associations

    • Controllers

  • Each method is a function that your application exposes
  • Controls the flow of the request
  • Exposes your data to views

    • Views

  • Show data to the user (in various format: html, csv, pdf..)

    • Routes

    It is a map between the HTTP request and the method of a controller

    Installation and Creation

    Installation :
    $ gem install rails

    Lets now create an app:
    $ rails new appname

    Lets try an app now

    Start an app :
    $rails s

    $ rails g controller home index contacts about

    Lets restart the server now
    and go to "localhost:3000/home/index" and contacts

    $ rails g scaffold blog title:string post:text

    $ rake db:migrate

    Restart server and open "localhost:3000/blogs"
